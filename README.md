# cloudSimGui
Simulation d'infrastructure Cloud avec CloudSim Plus
Prérequis
•	Java JDK 17
•	NetBeans IDE
•	CloudSim Plus 7.3.1
•	JavaFX SDK 17.0.8
Installation
Étape 1 : Cloner le projet

git clone https://github.com/djodakevin/cloudSimGui.git
cd cloudSimGui
 
Étape 2 : Configuration JavaFX Modifier le chemin dans nbactions.xml

Étape 3 : Build Maven

 mvn clean install
 
Exécution
Avec NetBeans :
1.	Ouvrir le projet
2.	Clic droit sur MainApp.java → Run File
